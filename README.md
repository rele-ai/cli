# Usage
<!-- usage -->
```sh-session
$ npm install -g @releai/cli
$ rb COMMAND
running command...
$ rb (-v|--version|version)
@releai/cli/0.1.0 darwin-x64 node-v15.2.1
$ rb --help [COMMAND]
USAGE
  $ rb COMMAND
...
```
